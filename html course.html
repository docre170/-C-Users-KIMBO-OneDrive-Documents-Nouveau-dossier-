Apprendre le HTML : Les Fondations du Web

HTML, ça veut dire HyperText Markup Language. Oublie le nom compliqué, retiens juste que c'est le squelette de toutes les pages web ! Quand tu vois du texte, des images, des liens sur un site, c'est grâce à HTML qu'ils sont structurés.

Imagine que tu construis une maison :

  • HTML, ce sont les murs, le toit, les portes... la structure de base.
  • CSS (qu'on verra après), c'est la peinture, la décoration, pour que ce soit joli.
  • JavaScript (plus tard !), c'est l'électricité, la plomberie, pour que ça devienne interactif.

À quoi ça sert concrètement ?

Avec HTML, tu vas pouvoir :

  • Créer des titres (comme celui que tu lis) avec les balises <h1>, <h2>, etc.
  • Écrire des paragraphes de texte avec la balise <p>.
  • Ajouter des images avec <img src="url_de_l_image" alt="description">.
  • Faire des liens vers d'autres pages avec <a href="autre_page.html">.
  • Et plein d'autres choses !

Un tout petit exemple

Voici à quoi ressemble un bout de code HTML très simple : 

            <!DOCTYPE html>
            <html>
            <head>
                <title>Ma Première Page</title>
            </head>
            <body>
                <h1>Salut le Monde !</h1>
                <p>Ceci est mon premier paragraphe en HTML.</p>
                <img src="https://placehold.co/200x100/3C2E69/E2DDFE?text=Mon+Image" alt="Exemple d'image">
            </body>
            </html>

Pas si sorcier, hein ? Le HTML, c'est surtout une question de balises qui s'ouvrent (<nomdelabalise>) et qui se ferment (</nomdelabalise>). On va explorer ça ensemble !

Les balises les plus courantes

  • <h1> ... </h1> à <h6> ... </h6> : Titres de différents niveaux
  • <p> ... </p> : Paragraphe de texte
  • <img src="..." alt="..."> : Image
  • <a href="..."> ... </a> : Lien cliquable
  • <ul>, <ol>, <li> : Listes à puces ou numérotées
  • <div> ... </div> : Division ou section de page
  • <span> ... </span> : Pour styliser une partie de texte
  • <br> : Saut de ligne

Structure d’une page HTML

Une page HTML commence toujours par <!DOCTYPE html> pour indiquer le type de document.
Ensuite, tout le contenu est placé entre <html> et </html>.
À l’intérieur, on trouve deux grandes parties :

  • <head> ... </head> : informations sur la page (titre, liens vers des fichiers CSS, etc.)
  • <body> ... </body> : tout ce qui s’affiche à l’écran (titres, textes, images, etc.)

Pour aller plus loin

  • Tu peux ajouter des attributs aux balises, par exemple <a href="https://exemple.com">Visiter</a>
  • Pour styliser ta page, tu utiliseras le CSS (voir le cours CSS !)
  • Pour rendre ta page interactive, tu découvriras JavaScript plus tard.

À toi de jouer !

Essaie de créer ta propre page HTML en t’inspirant de l’exemple ci-dessus.
Modifie le texte, ajoute une image différente, change le titre…
Plus tu pratiques, plus tu comprendras vite !

Astuce : Tu peux ouvrir ton fichier HTML dans un navigateur pour voir le résultat en direct.
